Welcome

Thank you for visiting the LIBR 289 ePortfolio website for Melissa Lafranchise.

Navigation

To navigate this site please select an option from the menu at left. The menu has been organized for easy access to the required components of the ePortfolio. Clicking on the Competencies option will reveal links to the 14 Core Competencies.

Content areas

This center area will always be used for the main content of the page. The right side bar will contain context-dependent quick links as appropriate for particular pages. It will be utilized heavily for the Core Competencies to display links to evidence for each Core Competency.

Process

To create this e-Portfolio, evidence that illustrated an understanding of each Core Competency was gathered from course assignments as well as work samples from current and past employers. A discussion of the key concepts comprising each competency was written as an introduction to the evidence. The subsequent description of each piece of evidence explained how the evidence supported competency in a given aspect of library and information science.
